【 資料管理 】

您可以將目前的「行程資料」與「待辦備忘」匯出成一個 JSON 檔案。 這個檔案可以備份您的資料，或傳送給同樣使用此工具的朋友，讓他們匯入。

使用說明

手機掃描上方 QR Code 開啟/安裝此 PWA 工具箱

計算器

  • 基本運算：直接輸入數字和運算符，然後按 = 計算結果。
  • 開方：輸入 n√x 格式（如 √25 或 3√27），然後按 = 計算。
  • 次方：輸入 x^n 格式（如 2^3），然後按 = 計算。
  • 百分比：輸入數字後按 % 轉換為百分比。
  • CE：清除最後一次輸入。
  • C：清除所有輸入。
  • ←：刪除最後一個字符。

幣值兌換

  • 輸入要兌換的金額。
  • 選擇原始貨幣和目標貨幣。
  • 點擊"兌換"按鈕查看結果。
  • 點擊"更新匯率"按鈕獲取最新匯率。

時區換算

  • 選擇或輸入本地時間。
  • 選擇原始時區。
  • 選擇目標時區。
  • 轉換後的時間會自動顯示在下方。

待辦備忘

  • 在輸入框中輸入新的待辦事項，按 Enter 鍵或點擊"加"按鈕來新增項目。
  • 單擊待辦事項可以選中它，背景會變為灰色表示已選中。
  • 使用 Ctrl/Cmd + 點擊可以選擇多個項目。
  • 使用 Shift + 點擊可以選擇一個範圍的項目。
  • 選中項目後，可以進行以下操作：
    • 切換重要：將選中的項目標記為重要或取消重要標記。
    • 切換完成：將選中的項目標記為已完成或取消完成標記。
    • 編輯：當只選中一個項目時，可以編輯其內容。
    • 刪除：移除選中的項目、已完成項目或所有項目。
  • 拖曳排序：
    • 點擊並按住任何待辦事項
    • 拖動它到您想要的位置
    • 釋放滑鼠按鈕來放置待辦事項
    • 拖動時會顯示半透明效果
    • 拖動到其他項目上方時會顯示綠色邊框提示
  • 重要的待辦事項會在左側顯示黃色標記。
  • 已完成的待辦事項會顯示刪除線。
  • 所有更改會自動保存，下次打開應用時仍然可以看到。
  • 每次操作後，選中狀態會自動清除，項目背景恢復正常。

行程資料

  • 預設為唯讀模式，雙擊內容區域即可進入編輯模式。
  • 編輯模式下可以使用以下功能：
    • 複製：複製整個行程資料到剪貼簿
    • 清除：清除所有行程資料
    • 粗體：將選中的文字設為粗體
    • 高亮：將選中的文字設為黃色背景
    • 紅色文字：將選中的文字設為紅色
    • 藍色文字：將選中的文字設為藍色
    • 黑色文字：將選中的文字設為黑色（可用於清除文字顏色）
    • 白色背景：將選中的文字設為白色背景（可用於清除背景色）
    • 加入待辦：將選中的文字加入待辦事項列表
    • 退出編輯：點擊 × 按鈕或點擊編輯區域外退出編輯模式
  • 所有更改會自動保存，下次打開應用時仍然可以看到。
  • 在編輯模式下，可以自由編輯、格式化文字，並將重要內容加入待辦事項。
  • 提示：要清除文字的格式，可以將文字設為黑色並使用白色背景。

語言翻譯

  • 預設源語言為繁體中文，目標語言為英文。
  • 在左側下拉選單選擇源語言。
  • 在右側下拉選單選擇目標語言。
  • 點擊中間的箭頭按鈕可以交換源語言和目標語言。
  • 在上方文本框中輸入要翻譯的文字。
  • 點擊右上角的喇叭圖標可以朗讀原文。
  • 點擊"翻譯"按鈕進行翻譯。
  • 翻譯結果會顯示在下方的文本框中。
  • 點擊翻譯結果右上角的喇叭圖標可以朗讀譯文。
  • 支持以下語言：
    • 中文（簡體）
    • 中文（繁體）
    • 英文
    • 日文
    • 韓文
    • 越南文
    • 泰文
    • 印尼文
    • 馬來文

資料管理

  • 您可以將目前的「行程資料」與「待辦備忘」匯出成一個 JSON 檔案。
  • 點擊「資料管理」( ) 分頁。
  • 匯出資料：點擊「匯出資料」按鈕，即可下載 `travel-tools-data.json` 檔案。
  • 分享檔案：您可以透過 Email、LINE 或其他通訊軟體，將這個檔案傳送給朋友。
  • 匯入資料：點擊「匯入資料」按鈕，選擇收到的 `.json` 檔案，即可載入朋友的資料或您自己的備份。請注意，匯入會覆蓋現有資料。
