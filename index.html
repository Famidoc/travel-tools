使用說明

手機掃描上方 QR Code 開啟/安裝此 PWA 工具箱

計算器

  • 基本運算：直接輸入數字和運算符，然後按 = 計算結果。
  • 開方：輸入 n√x 格式（如 √25 或 3√27），然後按 = 計算。
  • 次方：輸入 x^n 格式（如 2^3），然後按 = 計算。
  • 百分比：輸入數字後按 % 轉換為百分比。
  • CE：清除最後一次輸入。
  • C：清除所有輸入。
  • ←：刪除最後一個字符。

幣值兌換

  • 輸入要兌換的金額。
  • 選擇原始貨幣和目標貨幣。
  • 點擊"兌換"按鈕查看結果。
  • 點擊"更新匯率"按鈕獲取最新匯率。

時區換算

  • 選擇或輸入本地時間。
  • 選擇原始時區。
  • 選擇目標時區。
  • 轉換後的時間會自動顯示在下方。
  • 世界城市時鐘：下方網格提供 8 大旅遊熱門城市（東京、倫敦、紐約等）的即時時間，自動計算與台北時差並顯示日夜動態圖示（☀️/🌙）。

待辦備忘

  • 在輸入框中輸入新的待辦事項，按 Enter 鍵或點擊"加"按鈕來新增項目。
  • 單擊待辦事項可以選中它，背景會變為灰色表示已選中。
  • 使用 Ctrl/Cmd + 點擊可以選擇多個項目。
  • 使用 Shift + 點擊可以選擇一個範圍的項目。
  • 選中項目後，可以進行以下操作：
    • 切換重要：將選中的項目標記為重要或取消重要標記。
    • 切換完成：將選中的項目標記為已完成或取消完成標記。
    • 編輯：當只選中一個項目時，可以編輯其內容。
    • 刪除：移除選中的項目、已完成項目或所有項目。
  • 拖曳排序：
    • 點擊並按住任何待辦事項
    • 拖動它到您想要的位置
    • 釋放滑鼠按鈕來放置待辦事項
    • 拖動時會顯示半透明效果
    • 拖動到其他項目上方時會顯示綠色邊框提示
  • 重要的待辦事項會在左側顯示黃色標記。
  • 已完成的待辦事項會顯示刪除線。
  • 🎒 行李打包範本：點選輸入框下方「🎒 導入行李打包範本」按鈕，即可將必備的證件、外幣、充電設備、盥洗衣物等高頻行李清單一鍵載入到您的待辦清單中，方便在收拾行李時逐項打勾。
  • 所有更改會自動保存，下次打開應用時仍然可以看到。
  • 每次操作後，選中狀態會自動清除，項目背景恢復正常。

行程資料

  • 預設為唯讀模式，雙擊內容區域即可進入編輯模式。
  • 編輯模式下可以使用以下功能：
    • 複製：複製整個行程資料到剪貼簿
    • 清除：清除所有行程資料
    • 粗體：將選中的文字設為粗體
    • 高亮：將選中的文字設為黃色背景
    • 紅色文字：將選中的文字設為紅色
    • 藍色文字：將選中的文字設為藍色
    • 預設字色：將選中的文字恢復為預設顏色（可用於清除紅、藍色）
    • 預設背景：將選中的文字恢復為預設背景色（可用於清除高亮）
    • 加入待辦：將選中的文字加入待辦事項列表
    • 退出編輯：點擊 × 按鈕或點擊編輯區域外退出編輯模式
  • ✈️ 出海/出國倒數儀式：右下角常駐懸浮小飛機按鈕。設定出發日後，小飛機將變亮並呼吸抖動、顯示倒數天數 Badge。點選可彈出 Glassmorphism 彈窗查看大倒數計時與滑行軌跡，並可隨時修改或重置。
  • 🗂️ 多行程管理：點選頂部下拉選單可隨時切換不同行程。點選旁邊的「齒輪」按鈕可開啟行程管理，進行新增、刪除或重新命名。每個行程的待辦備忘、記帳明細與出發倒數皆獨立綁定與儲存。
  • 🎫 離線憑證錢包 (Travel Wallet)：行程編輯區下方特設離線憑證包。您可以上傳機票 QR Code、飯店確認信或護照照片，系統會在前端進行智慧體積壓制（最大上傳 4 張）。在國外沒有網路時，點選縮圖即可開啟全螢幕高解析燈箱模式，方便海關或登機驗證。
  • 所有更改會自動保存，下次打開應用時仍然可以看到。
  • 在編輯模式下，可以自由編輯、格式化文字，並將重要內容加入待辦事項。
  • 提示：要清除文字的格式，可以將文字設為預設字色並使用預設背景。

語言翻譯

  • 預設源語言為繁體中文，目標語言為英文。
  • 在左側下拉選單選擇源語言。
  • 在右側下拉選單選擇目標語言。
  • 點擊中間的箭頭按鈕可以交換源語言和目標語言。
  • 在上方文本框中輸入要翻譯的文字。
  • 點擊右上角的喇叭圖標可以朗讀原文。
  • 點擊"翻譯"按鈕進行翻譯。
  • 翻譯結果會顯示在下方的文本框中。
  • 點擊翻譯結果右上角的喇叭圖標可以朗讀譯文。
  • ⭐ 我的常用語（自訂對話庫）：在下方常用語模板中，可切換至「我的常用語」分類，點擊「新增」可永久保存常用詞句，點選快速氣泡即可一鍵翻譯並發音，亦支援單筆刪除。
  • 🤖 Gemini AI 雙通道翻譯：點選翻譯面板最上方「Gemini AI 翻譯設定」折疊欄位（點擊可展開或收合），即可輸入您專屬的 Gemini API 金鑰（僅安全儲存於您的本地瀏覽器中，不經過任何第三方伺服器，並提供密碼遮罩防窺）。設定後翻譯將由 Gemini 智能引擎驅動；若無設定或網路異常，會自動平滑降級至免費翻譯 API，保障翻譯服務不中斷。
  • 支持以下語言：
    • 中文（簡體）
    • 中文（繁體）
    • 英文
    • 日文
    • 韓文
    • 越南文
    • 泰文
    • 印尼文
    • 馬來文

旅遊記帳

  • 連續記帳與外幣折合：點選鈔票圖示進入記帳分頁，輸入消費備註與金額。支援多種外幣輸入，系統會依最新匯率自動折合為台幣儲存。
  • 記帳 Donut Chart 甜甜圈圖表：在總額下方，系統會自動繪製原生 Canvas 環形花費佔比圖，並依據花費金額由高到低以不同顏色排序各消費類別。
  • 消費明細編輯與修改：在消費明細列表中，點擊紀錄旁的「鉛筆」圖示，可開啟修改視窗一鍵修改消費品項、金額、類別或幣別，重算結果會即時同步至圓餅圖與明細中。
  • 記帳數據與當前行程完全獨立綁定，並隨行程匯出與備份。

資料管理

  • 您可以將目前的「行程資料」與「待辦備忘」匯出成一個 JSON 檔案。
  • 點擊「資料管理」( ) 分頁。
  • 匯出資料：點擊「匯出資料」按鈕，即可下載 `travel-tools-data.json` 檔案。
  • 分享檔案：您可以透過 Email、LINE 或其他通訊軟體，將這個檔案傳送給朋友。
  • 匯入資料：點擊「匯入資料」按鈕，選擇收到的 `.json` 檔案，即可載入朋友的資料或您自己的備份。請注意，匯入會覆蓋現有資料。
